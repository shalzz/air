# Airdrop checker

Python script to check multiple address for airdrop eligibility.

## How
Create and list the addresses you want to check in a text file `addrs.txt` in the root of
this directory.

## Support airdrops

* Uniswap
* 1inch
* Tornado
* Badger
* Furucombo
* StakeDAO
* Curve
* Digg
* PoolTogether
* Safe
* Hop Protocol
* Across Protocol
